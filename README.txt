AgriCircularToken è una semplice web Dapp che emette un token ERC-20 personalizzato (ACT) sulla testnet Sepolia. 
Consente agli utenti di accumulare token interagendo con lo smart contract (registrando le attività sostenibili) e di spenderli per ottenere badge, corsi o progetti ecologici.

-REQUISITI:

Per utilizzare la DApp è necessario:
	- avere installato l'estensione web MetaMask
	- essere connessi alla rete Sepolia tramite MetaMask
	- avere un minimo di ETH testnet nel wallet 

-AVVIO DELLA DAPP: ONLINE

È possibile accedere direttamente alla DApp tramite GitHub Pages al seguente link:
	https://valentinamilighetti.github.io/ACT-Dapp-DSB/

-AVVIO DELLA DAPP: LOCALE (con Node.js)

Se si desidera eseguire il progetto in locale, assicurarsi di avere installato Node.js, quindi:

	1. Aprire il terminale nella cartella del progetto
	2. Lanciare il comando:
   		npx http-server
	3. Aprire nel browser il link:
		http://localhost:8080
